Project Title: DormLife – Student Dormitory Web Application

Project Description:
DormLife is a modern web application designed specifically for students living in dormitories. The platform helps students manage daily dorm life by providing three main services: a laundry availability tracker, a student chat system, and a dorm marketplace where students can buy and sell items.

The application should have a clean, modern, and student-friendly design. The interface should be simple, intuitive, and mobile-responsive. The design should use a card-based layout, clear icons, and minimal colors to keep the interface easy to navigate.

Main Users:
Students living in a dormitory.

Main Goal:
Help students quickly check laundry machine availability, communicate with other dorm residents, and buy or sell items within the dorm community.
